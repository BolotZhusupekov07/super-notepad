and other features

## Installation

In the terminal use the package manager [pip](https://pip.pypa.io/en/stable/) to install modules.

```
pip install pipwin
```
```
pipwin install pyaudio
```
```
pip install SpeechRecognition
```
```
pip install gTTS
```
